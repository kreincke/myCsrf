Installation

Ja, es ist wirklich so leicht, myCsrf zu installieren: Sie können das Paket - wo immer Sie wollen - auspacken. Der erzeugte Ordner kann frei umbenannt und als Ihr neuer Projektordner verwendet werden.

Nach der Erfüllung der Voraussetzungen können Sie myCsrf downloaden

Pure Nutzung von myCsrf

Um selbst eine altphilologische / (geistes-) wissenschaftlich Arbeit zu erstellen, greifen Sie bitte auf das myCsrf Paket zurück. Das steht als tar.gz oder als zip Datei zum Download bereit. Beide Pakete enthalten die myCsrf-README Datei, die Benutzung Konfiguration näher erläutert.

  • http:///download.fodina.de/csrf/mycsrfr-1.1.tar.gz
  • http:///download.fodina.de/csrf/mycsrfr-1.1.zip

Teilnahme and der myCrsf Weiterentwicklung

myCrsf wird als Github Projekt veröffentlicht und gehostet:  https://github.com/kreincke/mycsrf.

So können Sie als Github-User das Project forken und Ihre eigenen Entwicklungslinie starten. Wir wären glücklich, wenn Sie uns Ihre Verbesserungen per email (karsten.reincke@fodina.de) zukommen lassen würden.

Oder Sie können direkt um eine Aufnahme als 'kollaborativer Teilnehmer' nachfragen. Sie sind herzlich willkommen!
